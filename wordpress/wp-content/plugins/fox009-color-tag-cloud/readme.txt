A beautiful, dynamic and colorful tag cloud.

== Description ==

The plug-in uses professionally designed tag colors as the background to present the tag cloud.

The tag cloud of the plug-in is fully responsive and correctly rendered in all browsers.

You can customize them with a lot of options and use them.

If you liked the plugin, please <strong>rate</strong> it.

This plugin is maintained by [Fox009](http://www.fox009.cn/), a WordPress resource site.

== Installation ==

1. Upload <strong>fox009-color-tag-cloud</strong> folder to the <strong>/wp-content/plugins/</strong> directory.
2. Activate it through the <strong>Plugins</strong> menu in WordPress management page.
3. Add the plugin's widget on the <strong>Appearance\Widgets</strong> page.
4. That's it.

== Frequently Asked Questions ==

= Does this plugin support categories? =

Yes, supporting categories is just like supporting tags.
